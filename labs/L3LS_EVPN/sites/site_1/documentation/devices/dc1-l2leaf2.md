## Interfaces

### Ethernet Interfaces

#### Ethernet Interfaces Summary

##### L2

| Interface | Description | Mode | VLANs | Native VLAN | Trunk Group | Channel-Group |
| --------- | ----------- | ---- | ----- | ----------- | ----------- | ------------- |
| Ethernet1 | L2_dc1-leaf3_Ethernet4 | *trunk | *10,20 | *- | *- | 1 |
| Ethernet2 | L2_dc1-leaf4_Ethernet4 | *trunk | *10,20 | *- | *- | 1 |
| Ethernet3 | SERVER_dc1-host2_eth1 | access | 20 | - | - | - |

*Inherited from Port-Channel Interface

#### Ethernet Interfaces Device Configuration

```eos
!
interface Ethernet1
   description L2_dc1-leaf3_Ethernet4
   no shutdown
   channel-group 1 mode active
!
interface Ethernet2
   description L2_dc1-leaf4_Ethernet4
   no shutdown
   channel-group 1 mode active
!
interface Ethernet3
   description SERVER_dc1-host2_eth1
   no shutdown
   switchport access vlan 20
   switchport mode access
   switchport
   spanning-tree portfast
```
